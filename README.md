# PDF to Excel Converter (Bank Statements)

A lightweight Streamlit app to extract transactions from **bank statement PDFs** and convert them into Excel.

### Features
- Upload PDF (max 200MB)
- Extract using:
  - **Table-based** (Camelot)
  - **Text-based** (Regex + pdfplumber)
- Download clean Excel file
